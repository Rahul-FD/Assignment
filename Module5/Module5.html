Answer

(a) embed vedio in a webpage =>

First of all go to the Youtube video which you want to embed.Then you have to click on that vedio.Form the list of share options , click on embed. Then you have to copy the HTML code form the box that oppears. Then you have to paste that copied code in your HTML file.

(b) embed audio in a webpage =>

The audio element works the same as the vedio element. The 'video' tag is used to embed video content in a document, such as a movie clip or other video streams. The 'video' tag contains one or more 'source' tags with different video sources. The browser will choose the first source it supports.

4. Canvas and SVG tags.

Answer

  • Canvas tag =>
    • This can, be used to draw graphs, combine photos, or create simple animations.

  • SVG tag =>
    • The svg tag defines a container for SVG graphics. SVG has several methods for drawing paths, boxes, circles, text, and graphic images.

    * Defrence in Canvas tag & SVG tag

    . SVG uses geometric shapes to render graphics, whereas Canvas uses pixels.

    . SVG gives better performance with smaller number of objects or larger surface. Canvas gives better performance with smaller surface or larger number of objects. SVG is vector based and composed of shapes
